# Project: ig-carousel-gen

Instagram carousel generator — Next.js 16 app with AI-powered slide generation.

## Stack
- Next.js 16 (Turbopack), React 19, TypeScript
- Tailwind CSS 4, Zustand, React Hook Form + Zod
- Sharp (image rendering), OpenAI API, Kie.ai API

## Deployment
- **Production URL:** https://ig-carousel-gen.vercel.app
- **GitHub:** https://github.com/philigranchik/ig-carousel-gen
- **Platform:** Vercel (auto-deploy on push to `master`)
- **Env vars on Vercel:** OPENAI_API_KEY, OPENROUTER_API_KEY, KIE_API_KEY, KIE_AI_ENABLED

## Commands
- `npm run dev` — dev server
- `npm run build` — production build
- `npx vercel --prod` — manual deploy to production
